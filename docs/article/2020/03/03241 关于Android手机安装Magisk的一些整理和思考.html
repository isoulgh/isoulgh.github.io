# 关于Android手机安装Magisk的一些整理和思考

2020.3.24

# 前言

有一种感概，通过今天的发现，总结一句话，“时代变了”，以前的通过V~上网，到今天的S~，V~的上网规则发生了截然不同，BGP、IPLC等等方式应运而生；再如Android手机从以前的获取Root权限衍生的各种一键刷机工具，到现在的Magisk，对现代技术的发展感慨万分。

# 一些见解

  1. Fastboot/Bootloader这两者个人暂时认为相同没有太大区别，Bootloader涉及到设备解锁问题，而fastboot所有设备均可进入。

  2. 感觉在现代很多手机更新会覆盖rec的情况下，一下两种命令没有什么区别了。

    fastboot flash recovery recovery.img

    fastboot boot recovery.img

  3. 贴一下两个命令，防止以后忘了

    adb reboot fastboot 命令进入fastboot模式

    adb reboot recovery　命令行进recovery模式
