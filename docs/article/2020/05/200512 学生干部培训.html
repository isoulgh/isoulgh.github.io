# 表彰公示文

  • 汇总表

    序号、班级（如“软件工程187”）、姓名、学号、（获奖情况【评比表之类的比较需要】）、联系方式

  • 通知

    下发在哪、交给谁、时间

  • 申报表

    1. 首行缩进
    2. 政治面貌
    3. 出生日期，如“1989.01”
    4. 字体字号，如“1.5/单倍行距 宋体/仿宋 5号字”
    5. 表格格式，如“占一张/两张相对完整的纸面”
    6. 汇总表，如“表名：信息工程学院——xxx 人数查漏等汇总后发送给各班干校对”

  • 公示

    1. 参考，如“参考学校以往公示文 其它学校的公示文/网上新闻”
    2. 公示日期，如“3-5个工作日”
    3. 落款，如“盐城师范学院信息工程学院学生工作办公室 共青团盐城师范学院信息工程学院总支委员会”
    4. 时间，如“二〇二〇年五月八日（靠右对齐）”

# 策划案

  1. 主题 形成思路 格式详细 节日 百度咨询 “学习强国”APP 共青团中央以及其它大学的公众号
  2. 格式，如“标题 3/4号字 宋体/黑体；正文 加粗 小四 仿宋 1.5倍行距/固定值20磅”
  3. 标题，如“一级：一、二、；（（一）（二））；二级：1.2.；三级（1）（2）；四级①②”
  4. 图片，插入图片前插入文本框，如“图1：图2：”，格式“五号宋体/黑体 加粗”

# 新闻稿

  1. 第一段直接写内容，不需要写目的背景
  2. 校对，一定要有审核人
